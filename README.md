
# Salve o Curintia — Entrega 1 (HTML5)

Projeto mínimo para atender aos requisitos:
- 3 páginas HTML (index, projetos, cadastro) com semântica, imagens e navegação.
- Formulário em `cadastro.html` com validação nativa HTML5 e máscaras (CPF, telefone, CEP).
- Estrutura de pastas organizada + assets otimizados (SVGs).

## Como abrir
Basta abrir `index.html` no navegador.

## Checklists sugeridos
- Valide cada HTML no W3C Validator.
- Verifique contraste/teclado (acessibilidade básica).
- Publicar no GitHub como **repositório público**.
